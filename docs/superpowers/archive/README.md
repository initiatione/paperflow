# Superpowers Archive

This directory keeps old implementation plans and specs out of the active `docs/superpowers/plans/` and `docs/superpowers/specs/` queues.

- `completed/`: plans/specs whose behavior is already represented by current code, docs, and tests.
- `superseded/`: plans replaced by a newer architecture decision; do not execute them as written.

Active plan/spec directories should contain only work that is still intended to guide implementation.
